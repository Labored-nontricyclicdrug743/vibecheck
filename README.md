## 🔍 What it checks

vibecheck looks for common issues that can create problems later. It focuses on checks like:

- API keys left in code
- weak or missing auth rules
- Supabase tables with RLS off
- missing rate limiting on endpoints
- CORS settings that allow too much access
- unsafe public data exposure
- config values that can open attack paths
- common OWASP-style mistakes

These checks help you spot problems early, before they become harder to fix
